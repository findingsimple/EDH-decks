# Secrets of Strixhaven - Pickup Guide

Cards from SOC (Commander), SOS (main set), and the Mystical Archive (SOA/SPG bonus sheet - 65 reprints with new Strixhaven art) worth picking up for existing decks. The main set is built around the five Strixhaven college color pairs (WB, UR, BG, RW, GU), but the Mystical Archive covers all colors.

## Riptide Rogue (UR Spellslinger) - BIGGEST WINNER

### From SOS
- Prismari, the Inspiration ({5}{U}{R}, Elder Dragon, ~$30) - All instants/sorceries have STORM
- Emeritus of Ideation // Ancestral Recall ({3}{U}{U} // {U}, ~$44) - 5/5 flyer with Ancestral Recall via Prepare
- Emeritus of Conflict // Lightning Bolt ({1}{R} // {R}, ~$12) - 2/2 first strike with Lightning Bolt via Prepare
- Resonating Lute ({2}{U}{R}, ~$7) - All lands tap for 2 mana (instants/sorceries only)
- Flashback ({R}, ~$11) - Give an instant/sorcery in graveyard flashback until end of turn
- Zaffai and the Tempests ({5}{U}{R}, ~$7) - Once per turn, cast an instant/sorcery from hand for free
- Thunderdrum Soloist ({1}{R}, ~$3) - Each instant/sorcery pings each opponent
- Great Hall of the Biblioplex (Land, ~$7) - Utility land that becomes a growing Wizard creature

### From SOC
- Veyran, Voice of Duality ({1}{U}{R}, Mythic) - Doubles all magecraft-style triggers
- Magma Opus ({6}{U}{R}, Mythic) - Big splashy finisher, can pitch for a Treasure
- Thunderclap Drake ({1}{U}) - Cost reducer + spell copier

### From Mystical Archive (SOA/SPG)
- Expressive Iteration ({U}{R}) - Premium UR card selection, look at 3 keep 2
- Pyretic Ritual ({1}{R}) - Fast mana for explosive storm turns
- Empty the Warrens ({3}{R}) - Storm token maker, tokens get prowess from Bria
- Flusterstorm ({U}) - Storm counterspell, excellent protection
- Brain Freeze ({1}{U}) - Storm mill wincon as an alternate threat
- Sleight of Hand ({U}) - Cantrip to chain spells
- Preordain ({U}) - Cantrip with scry 2
- Crackle with Power ({X}{X}{R}{R}{R}) - X damage finisher
