# Lands

Definitive record of which real (non-proxy) lands are in the collection and which deck currently holds each one. See `CLAUDE.md` for the maintenance rule (sync this file whenever a real land moves between decks).

## Conventions

- ✅ in the **Owned** column = real card owned. `✅×2`, `✅×3`, etc. = multiple real copies owned.
- **In Deck** = deck currently holding the real copy. Filenames map to display names (e.g. `ur-dragon.md` → `Ur-Dragon`). `Loose` = owned but not slotted into a deck. For multi-copy lands, comma-separate decks (e.g. `Ur-Dragon, Heavenly Inferno`).
- **Notes** flags duplicates (⚠ same card unmarked in two decks — needs review/verify whether intentional multi-copy) and recent moves.
- Empty Owned/In Deck = not in the collection (cycle wishlist).
- Basics, gates, vivid lands, bouncelands, and other commodity lands are **not** tracked here.
- **Row order within cycle tables** is by color pairing (ally pairs first: W/U, U/B, B/R, R/G, G/W; then enemy pairs: W/B, U/R, B/G, R/W, G/U), not alphabetical. This preserves cycle structure, keeps the same color pair on the same row across every table for fast cross-cycle scanning, and makes gaps in a cycle visually obvious. The flat lists at the bottom (Loose Collection, Conflicts to Resolve) have no inherent grouping and may be alpha-ordered if useful.

## Conflicts to Resolve

Lands appearing unmarked in 2+ decks. For each, decide:
- **Multi-copy owned** — mark `✅×N` in the cycle table and list both decks comma-separated in the In Deck column. No proxy update needed.
- **Single copy + proxy elsewhere** — pick the canonical deck, mark the other(s) `(Proxy)` in the deck file, and update this table.

**Trusted decks** (proxies correctly marked, so unmarked = real): all main-folder decks (Ur-Dragon, Riptide Rogue, Peace Offering, Heavenly Inferno, Plunder the Graves, Limit Break, Cunning Conqueror, Turtle Power, Bare Bones, Pink Pony Club, High-Perfect Morcant) + Shadow the Hedgehog (lands only — non-land cards in Shadow are not yet audited). Duplicates between two trusted decks = confirmed multi-copy.
